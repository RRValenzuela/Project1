The Curb Effect

Website

This is a mockup of a food delivery website that employs universal design in regards to color. The project is a great example of how important details such as color and contrast play a role in user experience and user interface design. It sheds light on the considerations that need to be made when creating universal designs. Co-Designed with Dario Dorella.
